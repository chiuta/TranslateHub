⚖ Legal & GDPR
TranslateHub does not collect, store or process personal data. Text entered in the translation form is sent directly to the Anthropic API (Claude) and is not retained by this site.

Legal basis (GDPR Art. 6)

Processing of user-entered text for translation purposes is carried out on the basis of Art. 6(1)(b) — performance of the service requested by the user — and Art. 6(1)(f) — legitimate interest in providing the requested functionality. No implicit or bundled consent is claimed; if you do not want your text sent to Anthropic, do not use the translation form.

What data TranslateHub processes

  • Text entered — sent temporarily to the Anthropic API for translation generation. Not stored on TranslateHub servers (there are none).
  • Uploaded files — processed exclusively in your browser (client-side). Only the extracted text is sent to Anthropic for translation.
  • API key (if provided) — stored in your browser's sessionStorage, cleared when you close the tab. Never transmitted to TranslateHub.
  • Interface language preference — stored in your browser's localStorage. Never transmitted.
  • Hosting access logs — the web host (e.g. GitHub Pages, Cloudflare Pages, or similar static host) may record standard access logs (IP address, timestamp, User-Agent) per its own policy. TranslateHub does not access or use these logs.

Third-country transfers (GDPR Chapter V)

By using this page you establish connections to the following non-EU services. Your IP address is technically transmitted to each of them:

  • Anthropic, Inc. (USA) — api.anthropic.com — receives your translation input. Per Anthropic's API policy, prompts and completions are retained for up to 30 days for abuse monitoring unless you use a Zero Data Retention agreement. See anthropic.com/privacy.
  • Google LLC (USA) — fonts.googleapis.com, fonts.gstatic.com — receives your IP to deliver web fonts. See policies.google.com/privacy.
  • Cloudflare, Inc. (USA) — cdn.jsdelivr.net — receives your IP only if you upload a .docx file (mammoth.js is lazy-loaded on demand). See cloudflare.com/privacypolicy.

These transfers rely on the standard contractual clauses and the EU–US Data Privacy Framework adequacy decision (July 2023). If you want zero third-party connections, self-host the fonts and mammoth.js locally.

Your rights (GDPR Chapter III)

  • Right of access (Art. 15)
  • Right to rectification (Art. 16)
  • Right to erasure — "right to be forgotten" (Art. 17)
  • Right to restriction of processing (Art. 18)
  • Right to data portability (Art. 20)
  • Right to object (Art. 21)
  • Right to lodge a complaint with your national supervisory authority (ANSPDCP in Romania, or your local DPA)

Because TranslateHub stores nothing on its own servers, rights 15, 17, 18 and 20 are satisfied trivially (there is nothing to access, erase, restrict or port). For data held by Anthropic, exercise your rights directly with them.

No tracking No data selling No cookies from this site Jurisdiction: Romania / EU
TranslateHub does not set any cookies of its own. There are no tracking, advertising or profiling cookies. No ads. No analytics.

Browser storage used by TranslateHub (not cookies)

  • sessionStorage — your Anthropic API key (if you provide one) is stored here and automatically cleared when you close the tab. Never transmitted to TranslateHub.
  • localStorage — your chosen interface language code (e.g. ui_lang_pref=ro) and cached UI translations for that language. Never transmitted.

You can inspect and clear these at any time via your browser's developer tools (Application → Storage).

Third-party connections that may set cookies

  • Google Fonts (fonts.googleapis.com, fonts.gstatic.com) — fonts are fetched from Google's CDN. Google may process your IP address and set its own cookies per its policy. To avoid this, use a blocking extension or self-host the fonts.
  • jsdelivr CDN (cdn.jsdelivr.net) — used only when you upload a .docx file to lazy-load mammoth.js. Operated by Cloudflare.
  • Anthropic API (api.anthropic.com) — API calls use header-based auth only, no cookies.

Legal basis

Strictly necessary technical storage does not require consent under the ePrivacy Directive 2002/58/EC. No analytical or marketing storage is used.

Technology stack

HTML5 · CSS3 · JavaScript ES2022 No frontend frameworks Single-file app
  • Claude Sonnet (Anthropic) — AI translation engine, accessed via public API. Model: claude-sonnet-4-20250514.
  • mammoth.js v1.6 — text extraction from .docx files, runs exclusively in the browser (client-side, CDN jsdelivr).
  • Google Fonts — Cormorant Garamond, Barlow, Barlow Condensed. Loaded from Google CDN.
  • Native Web APIs: Fetch API, Streams API, File API, Clipboard API, Blob API, DOMParser, execCommand (Rich Text), contenteditable.

Architecture & transparency

  • TranslateHub is a single-file HTML application. All source code can be inspected via "View Page Source" in the browser.
  • No proprietary backend. All AI calls go directly to api.anthropic.com.
  • File processing (.docx, .csv, .json, .po, .srt) is done entirely in the browser, with no upload to any TranslateHub server.
  • Long text translation uses an automatic chunking system with a 400-character context window between segments for coherence.

External dependencies
